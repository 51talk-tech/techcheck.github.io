Technical Check Process Guide

Please read and follow every step before starting.

⚠ Before You Start

If you encounter any errors, take a screenshot of the error and report it to the Technical Support Team using the AI Chatbot on your homepage.

Requirements

  • Use Microsoft Edge browser.
  • Windows Operating System is required.
  • LAN Cable connection is highly recommended.

Before Doing the Technical Check

  • Do not schedule lessons during or immediately after the check.
  • Restart your computer.
  • Turn OFF your modem for 3–5 minutes before reconnecting.
  • Use a LAN cable instead of WiFi.

Common Issues

Cannot Run Tech Tool

  • Use Windows Defender.
  • Disable or uninstall Avast, Norton, McAfee and other antivirus software.

Cannot Connect to Server

  • Temporarily disable your Windows Firewall.
  • Turn it back ON after using the tool.

Technical Check Results

The Technical Support Team reserves the right to overrule the results generated by the Tech Tool. The final evaluation will always depend on the Technical Support Team.

1

Visit the Technical Check Portal

Open Microsoft Edge and visit: https://51talk-tech.github.io/techcheck.github.io/

Read the guide carefully, then click I Agree.

Open Website
2

Download the Tool

A new tab will open. Click the Download button.

3

Open Downloads Folder

Wait until the download finishes. Click the folder icon.

4

Extract the ZIP File

Locate the ZIP file. Right click Extract Here or Extract All

If extraction fails, install WinRAR first.

Download WinRAR
5

Open the Extracted Folder

Open the newly created Tech Tool folder.

6

Run as Administrator

Right-click the Tech Tool. Select Run as Administrator.

Click Yes if prompted. If errors occur, repeat Step 4.

Support Portal
7

Windows SmartScreen

Click More Info then Run Anyway.

If you don't see this screen, proceed to Step 8.

8

Run the Technical Check

Enter your 51Talk Email and Password.

Select the purpose. Click Start Check.

If it runs longer than 5 minutes, close it and run again.

9

Save Your Result

Take a screenshot of the final result. You only have one attempt.

If the tool fails or results do not appear, contact Technical Support and submit a ticket using the AI Chatbot.

✔ You're Done!

Keep a screenshot of your Technical Check Result for verification. If your result is FAILED, follow all recommendations before requesting another attempt.
